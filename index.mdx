<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-bold text-gray-900 dark:text-white mb-3">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your Shopify store connected in minutes with these simple steps.
  </p>
  
  <Steps>
    <Step title="Install the Shopify App">
      Install the Ecomdrop IA Connector from the Shopify App Store or deploy your own instance using Docker.
      
      ```bash
      # Clone and install locally
      git clone https://github.com/Mercaline2024/Ecomdrop-ia-connector-2.git
      cd ecomdrop-ia-connector-2
      npm install
      ```
    </Step>
    
    <Step title="Configure Your API Keys">
      Set up your environment variables with your Shopify, Ecomdrop, and Dropi credentials.
      
      ```env
      # Shopify Configuration
      SHOPIFY_API_KEY=your_shopify_api_key
      SHOPIFY_API_SECRET=your_shopify_secret
      
      # Ecomdrop Integration
      ECOMDROP_API_KEY=your_ecomdrop_key
      
      # Dropi Integration
      DROPI_STORE_NAME=your_store_name
      DROPI_TOKEN=your_dropi_token
      ```
      
      <Note>
        Find detailed configuration instructions in the [Environment Setup](/getting-started/environment-setup) guide.
      </Note>
    </Step>
    
    <Step title="Connect Your Integrations">
      Navigate to the configuration page in your Shopify admin and connect your Ecomdrop and Dropi accounts. The app will validate your credentials and sync your flow configurations automatically.
    </Step>
    
    <Step title="Import Your First Product">
      Browse products from Dropi, select the items you want to sell, and import them to your Shopify store with customizable pricing and variants.
      
      <Tip>
        You can customize product names, descriptions, images, and pricing during the import process.
      </Tip>
    </Step>
  </Steps>
</div>
